Stylecodex Framework

This framework provide product card, which you can add in your project. Select your product card style from the below button.

Card-1 Implementation

CSS link
<link rel="stylesheet" href="https://github.com/RoookieDev/StyleCodex/blob/main/frameworkStyle.css">

Copy this CSS link in your Html file "head tag". And then paste below html in your body section

Html classes
<div class="card-1" >

    <div class="image-box" >
    <img src="image url" >
    </div>

    <div class="text-box" >
        <div class="title-box" >
        <h4>Product Name</h4>
        <h4>Price</h4>
        </div>
    </div>

    <div class="button-box" >
        <button>Buy Now</button>
        <button>Add Cart</button>
    </div>

</div>
